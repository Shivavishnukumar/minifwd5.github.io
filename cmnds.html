Linux Basic Commands

Command Description Usage
ls The ls command lists information about files and directories in the current directory. ls or ls -a (to list all files, including hidden ones)
cd The cd command is used to change the current working directory. cd directory_name
mkdir The mkdir command creates a new directory. mkdir directory_name
touch The touch command is used to create an empty file or update the timestamp of an existing file. touch file_name
pwd The pwd command prints the current working directory. pwd
rm The rm command is used to remove files or directories. rm file_name
rmdir The rmdir command is used to remove empty directories. rmdir directory_name
chmod The chmod command is used to change file permissions. chmod permissions file_name
sudo The sudo command is used to execute commands with elevated privileges. sudo command
cp The cp command is used to copy files and directories. cp source_file destination_file
mv The mv command is used to move or rename files and directories. mv source destination
cat The cat command is used to concatenate and display the content of files. cat file_name
clear The clear command is used to clear the terminal screen. clear
grep The grep command is used to search for patterns in text files. grep pattern file_name
less The less command is used to view the content of files one page at a time. less file_name
find The find command is used to search for files and directories based on various criteria. find path options pattern
tar The tar command is used to create, view, and extract files from tar archives. tar options archive_name files
zip The zip command is used to create, modify, and extract ZIP archives. zip options archive_name files
